// IEC 60534 — Liquid, non-choked
Cv = Q × (1/FP) × √(ρ/ρw / ΔP_eff)

// SI conversion constant: N1 = 0.0865

// Choked pressure drop limit:
ΔP_max = (FL × FP)² × (P1 − Ff × Pv)

// Liquid critical pressure ratio:

// Cavitation index:

// IEC 60534 — Gas/Vapor (N9 = 94.8 SI)
x = (P1 − P2) / P1 // pressure ratio
xchoked = γ/1.4 × xT // choked limit
if x > xchoked: use x = xchoked

Y = 1 − x / (3 × γ/1.4 × xT) // expansion factor

Cv = W / (N9 × P1 × Y) × √(T × Z / (x × MW))
